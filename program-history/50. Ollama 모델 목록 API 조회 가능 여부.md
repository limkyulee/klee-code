# 50. Ollama 모델 목록 API 조회 가능 여부

- 상태: 완료
- 작성일: 2026-06-29

## 질의

Ollama에 설치된 모델 목록을 YAML 설정이 아니라 `ollama list`처럼 API로 불러올 수 있는지 확인했다.

## 처리 내용

`LLMGateway.java`와 `application.yml`을 확인해 현재 모델 목록이 `klee.llm.models` 설정값을 기준으로 반환되고, 선택 모델 검증도 해당 YAML 목록을 기준으로 동작함을 파악했다. Ollama 공식 API의 로컬 모델 조회 엔드포인트인 `GET /api/tags`를 사용하면 설치된 모델 목록을 런타임에 조회할 수 있다고 안내했다.

## 변경 파일

- `program-history/50. Ollama 모델 목록 API 조회 가능 여부.md`

## 검증

코드 변경은 하지 않았고, 관련 파일 확인 및 공식 API 문서 확인으로 답변했다.

## 결과

Ollama 모델 목록을 YAML 대신 API로 조회하는 것이 가능하며, 프로젝트에서는 `LLMGateway.availableModels()`와 `isAllowedModel()`을 API 기반으로 바꾸면 된다고 설명했다.
