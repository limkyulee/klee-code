# 62. 스킬 프롬프트 한글화

- 상태: 완료
- 작성일: 2026-06-30

## 질의

`backend/src/main/resources/klee-prompts/skills/` 하위의 스킬 프롬프트도 한글로 작성해 달라는 요청이었다.

## 처리 내용

`review.md`, `explain.md`, `fix.md`, `test.md`의 영어 프롬프트를 확인한 뒤 같은 의미의 한글 지침으로 변경했다. 각 스킬의 목적과 우선순위는 유지하면서 Klee Code 내부 프롬프트가 한글 기준으로 관리되도록 맞췄다.

## 기능 추가/변경 결정 기록

- 요청 배경: 시스템 프롬프트에 이어 내부 스킬 프롬프트도 한글로 관리하려는 요청이었다.
- 선택한 기술/방식: 기존 Markdown 리소스 파일의 문구만 한글로 교체했다.
- 검토한 대안: 스킬 동작이나 파일 구조를 바꾸는 대안은 요청 범위를 벗어나 제외했다.
- 선택 이유: 프롬프트 리소스 구조와 slash skill 동작을 유지하면서 가장 작은 범위로 요청을 반영할 수 있다.
