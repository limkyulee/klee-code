# 25. 채팅 입력 줄바꿈 동작 확인

- 상태: 완료
- 작성일: 2026-06-23

## 질의

`ChatInput.tsx`에서 Enter는 채팅 전송, 실제 줄바꿈은 Shift+Enter로 동작하도록 적용되어 있는지 확인 요청.

## 처리 내용

`extension/webview-ui/src/features/chat/components/ChatInput.tsx`의 `handleKeyDown`과 `textarea` 연결을 확인했다. `event.key === 'Enter' && !event.shiftKey` 조건에서만 기본 입력을 막고 `submit()`을 호출하므로 Enter 단독 입력은 전송으로 처리된다. Shift+Enter는 해당 조건에 걸리지 않아 브라우저의 textarea 기본 동작인 줄바꿈이 유지된다. IME 조합 중에는 조기 반환하도록 처리되어 한글 조합 중 Enter 입력 오작동도 방지된다.

## 결과

Enter 단독 입력은 전송, Shift+Enter는 실제 줄바꿈으로 동작하도록 이미 적용되어 있음을 확인했다.
