# 51. Ollama 실제 설치 모델 조회 방식 설명

- 상태: 완료
- 작성일: 2026-06-29

## 질의

`application.yml`에 정의한 모델 목록이 아니라 실제 Ollama에 설치된 LLM 목록을 조회할 수 있는지, MCP 같은 방식을 써야 하는지 확인했다.

## 처리 내용

`LLMGateway.java`, `application.yml`, `LlmProperties.java`, `ModelResponse.java`, `LLMGatewayTest.java`, `build.gradle`을 확인했다. 현재 구현은 YAML의 `klee.llm.models`를 모델 목록과 허용 목록으로 사용하고 있음을 재확인했다. 실제 설치 모델 조회는 MCP 없이 Ollama HTTP API `GET /api/tags`를 백엔드에서 직접 호출하는 방식이 적합하다고 정리했다.

## 변경 파일

## 검증

코드 변경은 하지 않았고, 관련 구현 파일과 빌드 설정을 확인해 답변했다.

## 결과

실제 Ollama 설치 모델 목록은 API로 조회 가능하며, MCP는 이 요구사항에는 불필요하고 백엔드에서 Ollama API를 직접 호출하는 것이 맞다고 안내했다.
