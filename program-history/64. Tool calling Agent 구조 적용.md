# 64. Tool calling Agent 구조 적용

- 상태: 일부 완료
- 작성일: 2026-06-30

## 질의

현재 Klee Code 구조에 tool calling을 적용하는 방향으로 코드를 수정하고, 이번 작업은 히스토리에 남겨 달라고 요청했다.

## 처리 내용

백엔드에 `agent`, `tool`, `permission` 도메인을 추가했다. 기존 `/chat/stream`은 유지하고, 새 `/agent/stream` SSE 엔드포인트와 `/agent/tool-results` 결과 수신 엔드포인트를 추가했다. `AgentService`는 기존 `PromptAssemblyService` 결과에 tool calling 지침과 승인 모드를 붙여 중앙 `LLMGateway`로 요청하고, 모델이 `<klee_tool_call>{...}</klee_tool_call>` 형식으로 읽기 전용 tool을 요청하면 Extension Host에 `tool_call_requested` 이벤트를 보낸다.

v1 tool은 `read_file`, `search_files`만 허용했다. 백엔드 `ToolPolicyService`는 현재 읽기 전용 tool만 자동 실행 가능하게 제한하고, `ToolResultRegistry`는 Extension Host가 `/agent/tool-results`로 돌려주는 결과를 대기한다.

Extension Host에는 로컬 tool executor를 추가했다. `read_file`은 열린 VS Code 워크스페이스 내부 파일만 읽고, `search_files`는 워크스페이스 파일 경로를 검색한다. Webview의 `Ask for approval` 드롭다운 값은 `permissionMode`로 `SEND_MESSAGE` payload에 포함되며, Extension Host가 이를 `/agent/stream` 요청으로 전달한다.

설계 문서 `docs/DESIGN.md`, `docs/STRUCTURE_V2.md`에 agent/tool/permission 도메인과 `/agent/stream` 데이터 흐름을 반영했다.

## 기능 추가/변경 결정 기록

- 요청 배경: 사용자가 Ask for approval 드롭다운과 tool calling의 관계를 이해한 뒤, 실제 프로젝트에 해당 구조를 적용해 달라고 요청했다.
- 선택한 기술/방식: 기존 Spring Boot 백엔드 안에 `agent`, `tool`, `permission` 도메인을 추가하는 모듈러 모놀리스 방식으로 구현했다. 로컬 파일 접근은 백엔드가 직접 하지 않고 VS Code Extension Host의 local tool executor가 수행하도록 했다.
- 검토한 대안: 백엔드가 직접 파일을 읽는 방식은 사용자 로컬 워크스페이스 접근 경계와 맞지 않아 제외했다. 쓰기 tool과 명령 실행 tool까지 한 번에 추가하는 방식은 승인 UI와 위험 정책 범위가 커 v1에서 제외했다.
- 선택 이유: 현재 프로젝트의 “확장은 얇은 클라이언트, 정책과 감사는 백엔드” 원칙을 유지하면서도 사용자 로컬 파일 접근은 VS Code 권한 경계 안에 두는 방식이 유지보수성과 안전성 측면에서 적합하다.

## 검증

- `./gradlew compileJava`: 성공
- `npm run check-types`: 성공
- `npm run lint`: 성공
- `./gradlew test`: 실패. `PromptAssemblyServiceTest.includesInternalSkillForSlashCommand()`가 현재 `review.md` 리소스의 한국어 문구와 테스트 기대값 `"Lead with findings"` 불일치로 실패했다. 이번 tool calling 변경 파일과 직접 관련 없는 기존 테스트 기대값 문제로 확인했다.

## 결과

읽기 전용 tool calling 기반 agent v1 구조를 추가했다. Ask for approval 드롭다운 값은 이제 `permissionMode`로 요청에 포함되며, 실제 tool 실행 정책으로 연결될 수 있는 경로가 생겼다. 쓰기/명령 실행 tool과 실제 승인 팝업은 후속 단계로 남겼다.
