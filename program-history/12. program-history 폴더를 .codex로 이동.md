# 12. program-history 폴더를 .codex로 이동

- 상태: 완료
- 작성일: 2026-06-23

## 질의

사용자가 `program-history` 폴더도 `.codex` 하위로 이동해 달라고 요청했다.

## 처리 내용

`.codex/AGENTS.md`의 기록 경로를 `.codex/program-history`로 바꿨다.
기존 `program-history` 폴더를 `.codex/` 아래로 이동해 히스토리 파일들이 새 위치에서 계속 이어지도록 정리했다.
작업 트리의 다른 변경 사항은 그대로 유지했다.

## 기능 추가/변경 결정 기록

- 요청 배경: Codex 관련 프로젝트 지침과 기록 위치를 `.codex` 아래로 일관되게 모으려는 요청이었다.
- 선택한 기술/방식: 히스토리 저장 경로를 `.codex/program-history`로 고정하고, 기존 히스토리 파일을 그대로 새 위치로 이동했다.
- 검토한 대안: 루트 `program-history`를 유지하는 방안도 있었지만, 지침과 실제 저장 위치가 어긋나지 않도록 이동하는 쪽을 선택했다.
- 선택 이유: Codex가 참조하는 지침 경로와 실제 기록 위치를 한곳에 모으면 관리가 단순하고 혼선이 적다.

## 검증

- `mv program-history .codex/` 실행으로 폴더 이동을 확인했다.
- `find .codex/program-history -maxdepth 1 -type f | sort`로 파일들이 새 위치에 모두 존재하는지 확인했다.

## 결과

`program-history` 폴더를 `.codex/program-history`로 옮기고, Codex 지침도 새 경로를 보도록 정리했다.
