# 2. Codex 프로젝트 지침 기반 히스토리 설정

- 상태: 완료
- 작성일: 2026-06-23

## 질의

사용자는 Codex 대화 기록을 매번 별도로 요청해야 남기는 방식이 아니라, `.ai` 폴더 같은 설정 위치에 chat 기록 관련 지침을 넣어 자동으로 처리할 수 없는지 물었다.

## 처리 내용

- Codex가 별도 백그라운드 훅처럼 자동 실행되는 기능은 제공하지 않지만, 프로젝트 루트의 `AGENTS.md`를 통해 Codex가 매 턴 참고하는 저장소 지침을 둘 수 있다고 판단했다.
- 루트에 `AGENTS.md`를 추가했다.
- 해당 지침에 모든 Codex 작업 종료 시 `program-history` 아래에 다음 인덱스의 Markdown 기록을 작성하도록 명시했다.
- 기록 파일명, 기록 시점, Markdown 본문 형식, 민감 정보 제외 규칙을 함께 정의했다.

## 변경 파일

- `AGENTS.md`
- `program-history/2. Codex 프로젝트 지침 기반 히스토리 설정.md`

## 검증

별도 빌드나 테스트가 필요한 코드 변경은 아니다. 파일 생성과 내용 작성만 수행했다.

## 결과

앞으로 이 저장소에서 Codex가 루트 `AGENTS.md` 지침을 읽는 세션에서는 작업 완료 시 `program-history`에 대화 및 작업 기록을 남기도록 안내된다.
