# 43. 시스템 아키텍처 이미지 제공

- 상태: 완료
- 작성일: 2026-06-29

## 질의

서비스의 시스템 아키텍처를 이미지로 제공해 달라고 요청했다.

## 처리 내용

VS Code 확장, Spring Boot 백엔드, MongoDB, 단일 Ollama LLM 서버의 관계를 한눈에 볼 수 있는 SVG 다이어그램을 작성했다. 로그인 세션, 대화 세션, SSE 실행 세션, 다중 대화 상태 분리, LLM 서버의 실제 병렬 처리 경계를 함께 표시했다.

## 기능 추가/변경 결정 기록

- 요청 배경: 사용자가 주니어 백엔드 개발자도 이해할 수 있는 세션 및 다중 대화 구조를 시각 자료로 보고자 했다.
- 선택한 기술/방식: 생성형 래스터 이미지 대신 정확한 텍스트와 박스/화살표가 필요한 아키텍처 다이어그램에 적합한 SVG 파일을 `docs/system-architecture.svg`로 추가했다.
- 검토한 대안: PNG 생성 이미지는 텍스트 정확도와 유지보수성이 낮아 선택하지 않았다.
- 선택 이유: SVG는 확대해도 선명하고, Git에서 변경 이력을 추적하기 쉬우며, 문서에 직접 삽입하거나 브라우저에서 열기 쉽다.
