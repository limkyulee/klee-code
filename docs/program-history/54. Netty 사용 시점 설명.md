# 54. Netty 사용 시점 설명

- 상태: 완료
- 작성일: 2026-06-29

## 질의

이 프로젝트에서 Netty를 언제 사용하는지 확인해 달라는 요청이었다.

## 처리 내용

`backend/build.gradle`, `LLMGateway`, `OllamaApiConfig`, `ChatService`, `ModelController`를 확인했다. 애플리케이션 코드에서 Netty API를 직접 호출하지는 않으며, `spring-ai-starter-model-ollama`가 내부적으로 WebClient/Reactor Netty를 통해 Ollama 서버와 통신할 때 Netty가 사용된다고 판단했다.

## 변경 파일

- `program-history/54. Netty 사용 시점 설명.md`

## 검증

`rg`와 `sed`로 관련 의존성과 LLM 호출 경로를 확인했다. 코드 변경 및 테스트 실행은 필요하지 않았다.

## 결과

Netty는 직접 작성한 비즈니스 코드가 아니라 Spring AI Ollama HTTP 통신 경로에서 런타임 네트워크 엔진으로 사용된다고 설명했다.
