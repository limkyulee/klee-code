# 62. Agent 계층과 MSA 여부 설명

- 상태: 완료
- 작성일: 2026-06-30

## 질의

Agent/tool execution layer를 추가하는 구조 변경이 MSA 구조가 되는지 확인했다.

## 처리 내용

현재 구조와 앞선 agent/tool layer 설명을 기준으로, 계층 추가와 MSA 전환은 별개라고 정리했다. Agent orchestrator, permission policy, tool registry, audit 확장은 같은 Spring Boot 백엔드 안의 도메인/모듈로 둘 수 있으며 이 경우는 MSA가 아니라 모듈러 모놀리스 또는 레이어드 아키텍처 확장에 가깝다고 설명했다.

MSA가 되려면 agent, tool executor, audit, auth, LLM gateway 등이 독립 배포 단위와 네트워크 API, 데이터 소유권을 가져야 한다는 기준을 정리했다.
