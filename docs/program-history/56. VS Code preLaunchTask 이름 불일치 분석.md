# 56. VS Code preLaunchTask 이름 불일치 분석

- 상태: 완료
- 작성일: 2026-06-30

## 질의

VS Code에서 `Could not find the task 'extension: compile'.` 오류가 뜨는 이유를 확인해 달라는 요청.

## 처리 내용

`.vscode/launch.json`, `.vscode/tasks.json`, `extension/package.json`을 확인했다.
`launch.json`의 `preLaunchTask`는 `extension: compile`을 참조하지만, `tasks.json`에 등록된 실제 태스크 라벨은 `npm: compile - extension`이었다.
따라서 디버그 실행 전에 VS Code가 이름이 일치하는 태스크를 찾지 못해 오류가 발생한다고 판단했다.

## 변경 파일

- `program-history/56. VS Code preLaunchTask 이름 불일치 분석.md`

## 검증

설정 파일 조회로 원인을 확인했다. 별도 테스트나 빌드는 실행하지 않았다.

## 결과

오류 원인은 `launch.json`의 `preLaunchTask`와 `tasks.json`의 `label` 불일치라고 안내했다.
