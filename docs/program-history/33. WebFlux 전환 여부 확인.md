# 33. WebFlux 전환 여부 확인

- 상태: 완료
- 작성일: 2026-06-24

## 질의

백엔드가 WebFlux 로 전환되었는지 확인 요청.

## 처리 내용

`backend/build.gradle`, `ChatController`, `ChatService`, `WebConfig`를 확인했다. 빌드 의존성은 `spring-boot-starter-webmvc`와 `spring-boot-starter-webmvc-test`를 사용 중이며, 컨트롤러는 `SseEmitter`, 설정은 `WebMvcConfigurer` 기반이라 WebFlux 전환 상태가 아니라고 판단했다. `ChatService`의 스트리밍 반환 타입에만 Reactor `Flux<String>`이 사용되고 있다.

## 변경 파일

- `program-history/33. WebFlux 전환 여부 확인.md`

## 검증

`rg -n "webflux|spring-boot-starter-web|spring-boot-starter-webflux|WebClient|Mono|Flux|@RestController|RouterFunction" -S .`로 WebFlux 관련 의존성과 사용처를 확인했다. `sed`로 관련 파일 내용을 확인했다.

## 결과

현재 코드는 WebFlux가 아니라 Spring MVC 기반이며, 스트리밍 내부 구현 일부에만 `Flux`가 사용된다고 안내했다.
