# 73. Tool 도메인 분석

- 상태: 완료
- 작성일: 2026-07-01

## 질의

`backend/src/main/java/com/kleecode/backend/tool/` 패키지의 구조와 역할을 분석해 달라는 요청.

## 처리 내용

`tool` 패키지의 DTO와 서비스 5개를 확인하고, 이 도메인이 실제로 연결되는 `AgentService`, `AgentController`, VS Code extension의 local tool 실행 흐름까지 함께 추적했다. 백엔드는 도구 목록, 프롬프트 지시문, 정책 판단, 결과 대기를 담당하고 실제 파일 읽기와 검색은 extension에서 수행하는 구조로 정리했다. 유지보수 포인트로 백엔드와 extension의 도구 정의 중복, `PermissionMode` 미사용 정책 판단, 동기 대기 기반 실행 구조, 결과 상태 검증 부족을 확인했다.

## 검증

별도 테스트나 빌드는 실행하지 않았다. 코드 읽기와 호출부 검색으로 분석했다.

## 결과

Tool 도메인의 현재 책임, 실행 흐름, 장점과 개선 포인트를 사용자에게 설명했다.
