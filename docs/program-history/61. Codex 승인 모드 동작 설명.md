# 61. Codex 승인 모드 동작 설명

- 상태: 완료
- 작성일: 2026-06-30

## 질의

Codex의 chat input에 있는 승인 옵션이 실제로 영향을 준다면 동작이 어떤 형태로 달라지는지, 내부 시스템 프롬프트가 추가되는 방식으로 이해하면 되는지 확인했다.

## 처리 내용

OpenAI Codex 매뉴얼의 approvals, sandboxing, permissions 관련 섹션을 확인했다. Codex의 승인 옵션은 단순 시스템 프롬프트 추가가 아니라 샌드박스 모드, 승인 정책, 승인 리뷰어, 네트워크/파일 접근 경계와 연결되는 런타임 권한 설정으로 설명했다.

현재 프로젝트의 `ChatInput.tsx` 구현은 아직 해당 정책을 전달하지 않으므로 실제 Codex와 같은 동작을 만들려면 UI 선택값을 메시지 payload가 아니라 세션/런타임 권한 설정으로 연결해야 한다고 정리했다.
