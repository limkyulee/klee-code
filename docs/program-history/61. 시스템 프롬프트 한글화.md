# 61. 시스템 프롬프트 한글화

- 상태: 완료
- 작성일: 2026-06-30

## 질의

Klee Code 내부 시스템 프롬프트가 영어로 작성되어 있어 한글로 바꿔 달라는 요청이었다.

## 처리 내용

`backend/src/main/resources/klee-prompts/system.md`를 확인하고, 기존 영어 시스템 프롬프트를 같은 의미의 한글 문장으로 변경했다. 스킬별 프롬프트는 요청 범위가 시스템 프롬프트였으므로 변경하지 않았다.

## 기능 추가/변경 결정 기록

- 요청 배경: 내부 시스템 프롬프트를 한글 기준으로 관리하려는 요청이었다.
- 선택한 기술/방식: 기존 `system.md` 리소스 파일의 문구만 한글로 교체했다.
- 검토한 대안: 스킬 프롬프트까지 모두 한글화하는 대안이 있었지만, 요청 범위가 시스템 프롬프트였으므로 제외했다.
- 선택 이유: 프롬프트 리소스 구조를 유지하면서 가장 작은 범위로 요청을 반영할 수 있다.

## 변경 파일

- `backend/src/main/resources/klee-prompts/system.md`
- `program-history/61. 시스템 프롬프트 한글화.md`

## 검증

- `sed -n '1,80p' backend/src/main/resources/klee-prompts/system.md`: 한글 문구 반영 확인
- 문구 변경만 수행해 별도 빌드/테스트는 실행하지 않았다.

## 결과

Klee Code 내부 시스템 프롬프트를 한글로 변경했다.
